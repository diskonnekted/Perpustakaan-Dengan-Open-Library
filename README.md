# Perpustakaan Hybrid

A comprehensive Library Management System built with native PHP, designed to manage both physical book collections and digital resources (PDFs). This system integrates with OpenLibrary API for global book data retrieval and includes a built-in Content Management System (CMS) for library news and articles.

## Features

### Catalog Management
- **Physical & Digital Books**: Manage inventory for physical books and upload PDF files for digital collections.
- **Categorization**: Organize books by categories and shelves.
- **Search & Filter**: Advanced search functionality for local catalog.
- **Digital Reader**: Built-in full-screen PDF viewer for digital books.

### OpenLibrary Integration
- **Global Search**: Search for books globally using the OpenLibrary API.
- **Data Import**: Seamlessly import book metadata and covers from OpenLibrary to the local catalog.

### News & Content Module
- **News CMS**: Create, edit, and publish library news and articles.
- **Tags & Categories**: Organize content with flexible tagging and categorization.
- **Responsive Layout**: Modern grid layout for news listing and detailed article views.

### Admin Dashboard
- **Secure Authentication**: Protected admin panel.
- **Dashboard Overview**: Quick stats and management tools.
- **Settings**: Customizable library name, logo, and display options.
- **File Management**: Handle book covers, PDF uploads, and news images.

## Technology Stack

- **Backend**: PHP (Native)
- **Database**: MySQL / MariaDB
- **Frontend**: HTML5, Tailwind CSS, Alpine.js
- **External API**: OpenLibrary API

## Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/diskonnekted/Perpustakaan-Dengan-Open-Library.git
   ```

2. **Configure Web Server**
   - Place the project folder in your web server's root directory (e.g., `htdocs` for XAMPP).

3. **Database Setup**
   - Create a new MySQL database (e.g., `library_db`).
   - Import the database structure. You can use the provided setup scripts by accessing them in your browser:
     - `http://localhost/lib/setup.php` (Core tables)
     - `http://localhost/lib/setup_news.php` (News module tables)
   - Alternatively, update `config/database.php` with your database credentials.

4. **Configuration**
   - Edit `config/database.php` to match your database settings:
     ```php
     $host = 'localhost';
     $db   = 'library_db';
     $user = 'root';
     $pass = '';
     ```

## Usage

- **Public Interface**: Access the main page at `http://localhost/lib/` to browse the catalog and news.
- **Admin Panel**: Access `http://localhost/lib/admin/` to log in and manage the system.
  - Default credentials (if generated by setup): Check `setup.php` source or register a new admin if enabled.
